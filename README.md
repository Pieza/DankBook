# DankBook

A social network focus on memes.

## Getting Started
This application were develop with NodeJs and ReactJs. 

### Prerequisites

```
node 8.10 or up
```


### Installing
On the root folder run this command:
```
npm install
```
It will download all dependencies of the project.

## Running

To run the backend:
```
npm run dev
```

To run frontend:
```
npm run webpack
```
